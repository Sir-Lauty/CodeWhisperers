Doctor Stock: Gestor de Datos

Es un programa de control de stock el cual almacena los elementos que se agregen, ya sea de una empresa, supermercado, farmacia, etc.

Esta aplicación funciona agregando en los campos de texto el nombre y una cierta cantidad del elemento a agregar, también es posible seleccionarlos de la lista para editarlos (ya sea nombre o cantidad) o borrarlos. Es un programa fácil de usar, pensado para que pueda ser accesible incluso con conocimiento básicos de computación.

Se realizó con código Python, donde se usaron librerías como customTkinter para la realización del programa. Usamos MongoDB, una base de datos orientada a documentos guardados en estructuras de datos BSON.
